Project Details


Salient Features:

  1. This Http Server read the port Number, static image dir and template directory from configs.txt which is present in current directory.
  2. Home Page shows my Name, Subgroup, Roll number and image.
  3. It also has 2 hyper links, one to Project Description and one to Student details.
  4. Student details read Student data from a student database.
  5. It also consists a link to form which can be used to add new students.(using POST method)
  6. Any change in HTML files do need server restart. Just a refresh is need on browser.
